# PayrollTool
Small module to handle employee payments based on daily transactions

# Characteristics
* Employee assistance
* Transactions history
* Payroll Calculation
* Export calculation to excel

# Technologies
* Net Core 3.1
* ASP.Net MVC
* Entity Framework Core
* LINQ
* C#
* Javascript
* Jquery
* Bootstrap

# Database
* SQL Server
* Transact-SQL

# Third party libraries
* EPPlus
